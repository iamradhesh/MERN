HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page, and consists of a series of elements.


CSS is a language that describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work, because it can control the layout of multiple web pages all at once.

JavaScript is the programming language of HTML and the Web. JavaScript can change HTML content and attribute values.
